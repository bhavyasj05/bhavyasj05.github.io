## Getting Started

### Prerequisites
- Git
- Web browser

### Local Setup
1. Clone the repository
```bash
git clone https://github.com/yourusername/bhavya-portfolio.git
cd bhavya-portfolio
```

2. Open the project
- Simply open `index.html` in your web browser

## Git Workflow

### Initial Setup
```bash
# Initialize git repository
git init

# Add all files
git add .

# Commit initial files
git commit -m "Initial commit: Portfolio website setup"

# Create a new repository on GitHub
# Then add the remote
git remote add origin https://github.com/yourusername/bhavya-portfolio.git

# Push to GitHub
git push -u origin main
```

### Common Git Commands
```bash
# Check status
git status

# Add changes
git add .

# Commit changes
git commit -m "Descriptive commit message"

# Push to remote
git push origin main
```
