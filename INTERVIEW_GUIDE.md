### **"How did you deploy this project?"**

**Answer:**
"I deployed the portfolio using a modern CI/CD (Continuous Integration/Continuous Deployment) workflow:

**Step 1: Version Control**
- Initialized a Git repository
- Committed all project files
- Pushed to GitHub repository

**Step 2: Hosting Platform**
- Connected my GitHub repository to Netlify
- Netlify automatically detects changes and deploys

**Step 3: Configuration**
- Set up `netlify.toml` for build settings
- Configured security headers
- Enabled automatic HTTPS

**Step 4: SEO Setup**
- Submitted sitemap to Google Search Console
- Verified site ownership
- Optimized for search engine indexing

**Result:** Now whenever I make changes and push to GitHub, Netlify automatically rebuilds and deploys the updated site within 1-2 minutes. This ensures my portfolio is always up-to-date."
