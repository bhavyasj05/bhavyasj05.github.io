## Step 1: Submit to Google Search Console (MOST IMPORTANT!)

### 1.1 Go to Google Search Console
1. Visit: **https://search.google.com/search-console**
2. Sign in with your Google account (use the same account you use for Gmail)

### 1.2 Add Your Property
1. Click **"Add Property"** button
2. Select **"URL prefix"** option
3. Enter your URL: **`https://bhavyasj-portfolio.netlify.app`**
4. Click **"Continue"**

### 1.3 Verify Ownership
You have **3 options** to verify:

#### Option A: HTML Tag (Easiest - Recommended)
1. Google will give you a meta tag like:
   ```html
   <meta name="google-site-verification" content="YOUR_CODE_HERE" />
   ```
2. Copy the `content` value
3. I'll add it to your HTML files for you - just send me the code!

#### Option B: HTML File Upload
1. Download the HTML file Google provides
2. Upload it to your Netlify site root folder
3. Deploy the site

#### Option C: Domain Name Provider
1. If you have a custom domain, verify through DNS

### 1.4 Submit Sitemap
After verification:
1. Go to **"Sitemaps"** in the left menu
2. Enter: **`sitemap.xml`**
3. Click **"Submit"**
4. Wait for Google to process (usually 1-2 days)

### 1.5 Request Indexing
1. Go to **"URL Inspection"** tool (top search bar)
2. Enter: **`https://bhavyasj-portfolio.netlify.app/`**
3. Click **"Request Indexing"**
4. Repeat for these pages:
   - `https://bhavyasj-portfolio.netlify.app/about.html`
   - `https://bhavyasj-portfolio.netlify.app/skills.html`
   - `https://bhavyasj-portfolio.netlify.app/projects.html`
   - `https://bhavyasj-portfolio.netlify.app/contact.html`
